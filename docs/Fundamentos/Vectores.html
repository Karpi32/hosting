Vectores

Los vectores son arreglos que contienen solo una dimensión y las matrices 2 o más dimensiones. Los Arreglos se utilizan para guardar un conjunto de cambiantes, que sean del mismo género de apunte, y todas estas bajo un mismo nombre.

VECTORES

Arreglo Simple

Se trata de un espacio en la memoria donde se almacenan datos del mismo tipo, pueden ocupar el espacio que se desea y es muy util para guardar e ingresar datos en un mismo vector para luego ser leido por el sistema.

EJEMPLO:

A = [1,2,3,4,5,6,7,8,9]
-Vector simple con datos predeterminados

Arreglo matriz

Una matriz es similar a un arreglo, la diferencia es que este almacena varios tipos de datos como si en una tabla se tratace, estan van definidas en un orden determinado como: [1][2] Esto significa que el objetivo esta en la columna 1 y en la fila 2.

EJEMPLO:

import numpy as np

A = np.array([[1,4,5,12],[-5,3,7,1],[2,7,5,9]])
