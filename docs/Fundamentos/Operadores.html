Operadores

Estos operadores nos ayudan a realizar operaciones Matematicas y Aritmeticas dentro del software para agilizar el proceso de calculo y mostrar los datos

¿Que son operadores?

La sintaxis es sencilla y facil de comprender, basta con establecerlos de acuerdo a las necesidades que se esten buscando y se pueden añadir directamente en las variables o en una condicion.

1

SUMA, RESTA, MULTIPLICACION Y DIVISION: Estas son las mas sencillas de codificar ya que solo es necesario establecer la operacion que se desee hacer y colocar el signo respectivo para la operación
EJEMPLO:
Suma = 10 + 5
Division = 100 / 10

2

Mayor y Menor Que: Estos operadores son utilizados para comparar dos catidades, se utiliza comunmente en condiciones IF
EJEMPLO:
IF Variable1 < Variable2:
print("Hola mundo")

3

Operadores Especiales: Este tipo de operadores son especiales en el lenguaje de programación, ya que nos ayudan a establecer algunas condiciones que en la vida real se presentan tales como:

4

Mayor o Menor Igual Que: Es una variante del operador comun de Mayor o Menor Que, pero su diferencia es que cuando compara los valores estos buscan ser iguales o mayores (pueden ser menores tambien) al valor resultante esto ayuda para tomar en cuenta el dato frontera y establecer un rango mas preciso.
EJEMPLO:
IF A <= B:
Print("La variable A es menor o igual al valor de B")

5

Igual o Identico: Este operador se encarga de establecer un valor exacto cuando se comparan las cantidades, es decir que el valor a revisar debe ser exactamente al valor de la condicion.
EJEMPLO:
IF C == 10:
Print("El valor de C es exactamente 10")

6

Distinto o Diferente: Con este operador podremos descartar otro valor, quiere decir que si la condicion es distinta al valor que estamos evaluando esta se cumplira.
EJEMPLO:
IF x != 5:
Print("El valor de x es distinto de 5")
