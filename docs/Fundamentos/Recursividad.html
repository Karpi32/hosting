Recursividad

La recursividad consiste en funciones que se llaman a sí mismas, evitando el uso de bucles y otros iteradores.
Basicamente es una herramienta muy util para ahorrar espacio y codigo, y resulta una de las tecnicas mas usadas para agilizar el trabajo

RECURSIVIDAD

Variable recursiva

Se trata de dos variables que se identifican por el mismo nombre. Se podría renombrar la variable x del segundo bloque por y, sin cambiar en nada la ejecución del programa.

Funcion recursiva

Dicho de otra manera, una función recursiva se resuelve con una llamada a si misma, cambiando el valor de un parámetro en la llamada a la función.

Ejemplo de Fibonacci:

def fibonacci(n):
       if n == 0;
              return 0
       elif == 1:
          return 1
     else:
          return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(10))
