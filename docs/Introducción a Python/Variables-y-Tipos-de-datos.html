Variables

Una variable es un espacio en un sistema de almacenaje que recibe un identificador y contiene información conocida o desconocida.

Primero se coloca el nombre de la variable y seguido del operador "=" y los datos que va a almacenar. Dependiendo el dato que se almacena, Python definirá el tipo de variable.

Si la variable a almacenar es texto, esta debe ir entre comillas.

Los 3 tipos de variables más comunes son: númericas, texto y booleanas

print

print() es una función, por lo que  el contenido siempre debe estar entre paréntesis. Cuando print() incluye una expresión, esta es evaluada antes de ser presentada en pantalla.

asignacion
Operador Función Ejemplo Salida
= Igual x = 1+1 x=2
+= El primer elemento es igual a la suma del primer elemento con el segundo a+=1 a=a+1
-= El primer elemento es igual a la resta del primer elemento con el segundo. a-=1 a=a-1
/= El primer elemento es igual a la división del primer elemento con el segundo. a/=5 a=a/5
%= El primer elemento es igual a el módulo del primer elemento con el segundo. a%=5 a=a%5
**= El primer elemento es igual al resultado del exponente del primer elemento con el segundo c**=2 c=c**2
*= El primer elemento es igual a la multiplicación del primer elemento con el segundo. c*=2 c=c*2

Operador Función
