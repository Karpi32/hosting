Listas 

Se define una lista como una zona de almacenamiento contigua donde podemos almacenar numerosos y diferentes tipos de datos, la misma puede editarse, modificarse mediant e   método s siendo  mutable.
Podemos decir también que las listas son datos estructurados almacenados en una variable.

Cómo crear una lista en Python

Declaramos una variable y almacenamos la lista con sus elementos dentro de corchetes:

En la lista "letras", los elementos van dentro de comillas, ya que son de tipo string, es decir de texto.

Para definir una lista vacía, únicamente debemos colocar los corchetes.

Índices-------->    0              1              2             3            4

Los índices identifican el elemento dentro de la lista, en Python el primer índice siempre es 0.
Para la lista "letras" el elemento a se encuentra en la posición 0, el elemento c en la posición 2, y el elemento e en la posición 4

Cómo acceder a una lista en Python

Para acceder a un elemento específico de la lista se debe nombrar la lista seguido de corchetes y dentro de estos la posición o índice del elemento requerido.

Cortar una lista en Python

Para especificar el rango de índices que queremos mostrar, se nombrar la lista, seguida de corchetes y el rango con ":".

Si deseamos imprimir toda la lista excepto un elemento se específica el elemento con un "-".

Si queremos obtener toda la lista desde algún índice, se pone el índice, seguido de ":".

Modificar Listas

Método append:
Permite agregar un elemento al final de una lista en python, se coloca el nombre de la lista a modificar seguido de un punto, el método (append) y luego entre paréntesis el elemento a agregar.
